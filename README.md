# hungry-capybaras

## Overview
This repository contains a Python script that runs the arcade game _Hungry Capybaras_ and its assets. The game leverages pygame to render assets and detect object collisions. Game entities and interactions are handled using a class-based architecture.

## Repository Structure
```text
hungry-capybaras/
├─ sprites/
├─ scripts/
│  ├─ run_hungry_capybara.py
```

## Requirements
  1. pygame>=2.6
